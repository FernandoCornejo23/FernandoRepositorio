Estilos embebidos en las etiquetas HTML

usar estilos CSS embebidos en las etiquetas de estilo. NO es nada recomendable por la dificultad que implica realizar un cambio a posterior.

Esta forma de incluir CSS directamente en los elementos HTML solamente se utilizan en determinadas situaciones en las que se debe incluir un estilo muy especifico para un solo elemento concreto.
